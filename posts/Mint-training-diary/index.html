Mint调教日记

发表于2023-01-17|计算机学习心得

Linux Mint is a free Linux-based operating system designed for use on desktop and laptop computers.
Mint官网

基础设置

  • 将home目录语言改为英语
    1. export LANG=en_US
    2. xdg-user-dirs-gtk-update
  • 同步与Windows的时间：sudo timedatectl set-local-rtc true
  • 自动挂载Windows下的硬盘
    1. 关闭Windows快速启动
    2. 查看硬盘情况：df和blkid
    3. 打开配置文件：sudo xed /etc/fstab
    4. 在最后一行添加挂载：UUID=C0F89003F88FF646 /mnt/windowsdisk ntfs rw
  • 缩放比例/字体字号/电源管理/鼠标速度/主题（应用程序：Mint-Y-Blue）/驱动/DNS
  • git相关
    • 设置全局name和email
    • 设置git默认main分支：git config --global init.defaultBranch main
    • 设置git不显示中文：git config --global core.quotepath false
    • 添加SSH
      • 生成SSH（如果存在可以不用生成）：ssh-keygen -t rsa -C komaeka@163.com
      • 启动ssh-agent：ssh-agent bash
      • 将SSH私钥添加到ssh-agent：ssh-add ~/.ssh/id_rsa
      • 将公钥粘贴到GitHub
      • 验证是否设置成功：ssh -T git@github.com

软件安装

  • 安装字体：Comic Sans MS，Microsoft YaHei，JetBrains Mono
  • 安装grub-customizer设置开机启动项
  • 安装：vscode/clash/QQ/百度网盘/QQ音乐
    • QQ音乐：为防止闪退，安装好后sudo xed /usr/share/applications/qqmusic.desktop，第三行改为Exec=/opt/qqmusic/qqmusic --no-sandbox %U
  • 安装nodejs
    1. 下载并解压
    2. 将bin文件夹添加到环境变量，打开/etc/profile.d/myenvironment.sh，写入以下内容export PATH=$PATH:/home/komaeka/application/nodejs/bin
    3. source /etc/profile
