MySQL8.0的下载与安装

发表于2021-08-01|计算机学习心得
|字数总计:344|阅读时长:1分钟

下载

  1. 进入官网的下载页面

  2. 点击下图中的链接

  3. 可以选择上边的 Community Server ，那样会下载压缩包，这里我选择下边的 Installer for Windows ，下载的是安装包

  4. 点击下方的 Download

  5. 点击下方的 No thanks, just start my download. 开始下载

安装

  1. 根据需要选择安装，这里我选择Server only，点击Next

    Developer Default：默认，安装所有开发需要的部分
    Server only：仅安装MySQL数据库服务器
    Client only：仅安装MySQL数据库客户端
    Full：完全安装
    Custom：自定义安装

  2. 点击Execute，等它执行完毕，点击Next

  3. 直接点击Next

  4. 默认配置即可，直接点击Next

  5. 默认强密码，直接点击Next

  6. 上方设置root账户密码后点击Add User，添加用户，可以用同一个密码，防止记不住，点击OK后点击Next

  7. 默认配置即可，直接点击Next

  8. 点击Execute，等全部打上勾后，点击Finish

  9. 直接点击Next

  10. 直接点击Finish完成安装

测试

  1. 键入WIN+R，输入cmd，点击确定打开命令行

  2. 移动到mysql的bin文件夹下
    cd C:\Program Files\MySQL\MySQL Server 8.0\bin

  3. 输入mysql -u用户名 -p密码

  4. 输入select version();查看mysql版本，成功显示版本信息，安装完成
