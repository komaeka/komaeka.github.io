Brownie框架

发表于2022-11-24|计算机学习心得

Brownie is a Python-based development and testing framework for smart contracts targeting the Ethereum Virtual Machine.
Brownie文档

安装Brownie：pip install eth-brownie

框架结构

  • 初始化项目：brownie init

  • 初始化项目之后，文件夹会生成一下几个文件

    • build：存放对智能合约进行编译后产生的文件
    • contracts：存放智能合约源代码
    • interfaces：…
    • reports：…
    • scripts：存放调用智能合约的脚本
    • tests：存放调用智能合约的测试脚本

配置文件

先在项目根目录创建brownie-config.yaml，有关brownie的所有配置都可以在其中进行配置

要引用brownie-config.yaml中的配置参数，需要先from brownie import config，然后应用索引的方式config['networks'][network.show_active()]

编译

编译命令：brownie compile
对智能合约代码进行改动后都要重新进行编译

执行脚本

执行脚本放在scripts目录下

执行脚本的命令：brownie run scripts/deploy_lottery.py

测试

测试文件放在tests目录下

文件命名规则为：test_***.py，如test_simple_storage.py，函数命名规则也如此

测试分为三个步骤：Arrange，Act，Assert，一个测试函数可以进行两次assert
