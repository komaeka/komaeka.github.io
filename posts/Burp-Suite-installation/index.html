Burp Suite安装及浏览器代理设置

发表于2022-03-10|计算机学习心得

准备工作

  1. 下载安装java并配置java环境变量（我这里安装的是java16.0.2）
  2. 去Burp Suite官网下载Burp Suite专业版的JAR包
  3. 下载BurpLoaderKeygenCn注册机
  4. 将Burp Suite的jar包和BurpLoaderKeygenCn的jar包放在一个同文件夹

安装流程

  1. 在当前文件夹打开终端，输入java --illegal-access=permit -Dfile.encoding=utf-8 -javaagent:BurpLoaderKeygenCn.jar=cn -noverify -jar burpsuite_pro_v2022.2.2.jar（根据你下载的Burp Suite文件输入相应的文件名）
  2. 接下来会弹出一个让你输入许可证的窗口（以下简称Burp窗口），点击 Receive ，然后打开 BurpLoaderKeygenCn.jar 文件（以下简称注册机），将 License 的内容复制到Burp窗口里边，点击 Next ，并选择 Manual activation
  3. 然后点击Burp窗口里边的 Copy requests ，将其粘贴到注册机的 Activation Request 里边，然后再将注册机生成的 Activation Response 复制下来，点击Burp窗口的 Paste response ，点击Next即可完成注册（我注册过了，就没有图了）
  4. 最后记得打开 BurpLoaderKeygenCn.jar ，勾选上 CN （汉化）， Auto Run （自动运行），这样以后每次点它就能直接以汉化方式启动Burp Suite了

浏览器代理设置

  1. 安装Firefox（个人喜好）
  2. 在Firefox链接栏里输入http://burp/，点击右上角 CA Certificate 下载证书
